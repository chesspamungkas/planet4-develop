
# Requirements

- docker
- docker-compose
- node
- nvm
- wp-env

# Before anything

- Clone this repo
- Set node version
```console
> nvm use
```

# Installation

```console
> npm run env:install
```

# Clean up

```console
> npm run env:clean
```
